# deyvinteterko.uk

My personal portfolio website built from scratch to showcase my projects and experiences as a Computer Science student.

**Live Website:** https://deyvinteterko.uk

## About

This website acts as my online portfolio, where I document what I am learning and my projects I currently work on which shape my career as a aspiring Computer Scientist.

Rather than using a website builder, I chose to develop the site myself to improve my front-end development skills and have complete control the design and functionality of my website.

## Built With

- HTML5
- CSS
- JavaScript

## Features

- Responsive design
- Project showcase
- About Me page
- Contact links
- Clean, lightweight interface

## Future Plans

I'm continuously improving the website by experimenting with new features and refining the design.
